# windows-sysadmin-shortcuts

Atajos, comandos y un par de scripts en PowerShell que uso para administrar Windows sin pasar tres veces por el mouse.

Esto lo armo para mi referencia personal y para compartir con gente del laburo. Si te sirve, dale.

## Que va a haber aca

- consolas .msc que abro a diario
- comandos del Win+R utiles
- scripts PowerShell para tareas repetitivas
- atajos de teclado del SO y de PSReadLine
