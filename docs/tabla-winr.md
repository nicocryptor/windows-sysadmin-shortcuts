# Tabla extendida de comandos Win+R

La tabla corta esta en [atajos/win-r.md](../atajos/win-r.md). Esto es para
los comandos menos conocidos que igual aparecen una vez por mes.

## Diagnostico avanzado

| Comando | Que hace |
|---|---|
| `verifier` | Driver Verifier Manager. Para tirar al piso un driver sospechoso. |
| `wercon` | Vistas de Reportes de problemas (Win 7, sigue en 10/11) |
| `eudcedit` | Editor de caracteres privados |
| `iexpress` | Empaquetar un .exe self-extracting |
| `mblctr` | Mobility Center (laptops) |
| `osk` | Teclado en pantalla |
| `magnify` | Lupa |
| `narrator` | Narrador |

## Redes y conexiones

| Comando | Que hace |
|---|---|
| `cliconfg` | SQL Server Client Network Utility (clasico) |
| `iscsicpl` | Iniciador iSCSI |
| `wf.msc` | Firewall avanzado |
| `lpksetup` | Instalar/desinstalar paquetes de idioma |
| `optionalfeatures` | Activar/desactivar caracteristicas de Windows (igual que appwiz.cpl) |

## Seguridad y control de cuentas

| Comando | Que hace |
|---|---|
| `netplwiz` | Cuentas de usuario avanzadas, donde se quita el "Los usuarios deben escribir su nombre" |
| `useraccountcontrolsettings` | Configuracion del UAC |
| `credwiz` | Asistente para backup/restore de credenciales |
| `secpol.msc` | Politicas de seguridad local |
| `gpedit.msc` | Politicas de grupo local |

## DCOM, COM, WMI

| Comando | Que hace |
|---|---|
| `dcomcnfg` | DCOM Config |
| `comexp.msc` | Component Services |
| `wbemtest` | Test de WMI a mano |
| `wmimgmt.msc` | Configuracion del repositorio WMI |

## Diagnostico de memoria y hardware

| Comando | Que hace |
|---|---|
| `mdsched.exe` | Memory Diagnostic, te pide reiniciar |
| `chkdsk /scan` | Chequeo de disco online |
| `winsat formal` | Genera reporte de Windows Experience (CLI, no GUI) |

## URIs de Configuracion (ms-settings:)

En Win 10/11 muchas paginas de Settings se abren directo con `Win+R` o
`start ms-settings:XYZ`.

| URI | Donde te lleva |
|---|---|
| `ms-settings:windowsupdate` | Windows Update |
| `ms-settings:about` | Acerca de |
| `ms-settings:network-status` | Estado de red |
| `ms-settings:network-wifi` | Wi-Fi |
| `ms-settings:network-ethernet` | Ethernet |
| `ms-settings:bluetooth` | Bluetooth |
| `ms-settings:display` | Pantalla |
| `ms-settings:storagesense` | Storage Sense |
| `ms-settings:powersleep` | Energia y suspension |
| `ms-settings:privacy` | Privacidad |
| `ms-settings:windowsdefender` | Defender |
| `ms-settings:signinoptions` | Opciones de inicio de sesion |
| `ms-settings:troubleshoot` | Solucionar problemas |
| `ms-settings:taskbar` | Barra de tareas |

## Carpetas con shell:

Esto va en cualquier dialogo de "abrir" o en `Win+R`.

| Comando | Donde te lleva |
|---|---|
| `shell:profile` | Tu perfil C:\Users\xxx |
| `shell:cookies` | Cookies |
| `shell:cache` | Cache de IE/Edge |
| `shell:history` | Historial |
| `shell:fonts` | Fuentes |
| `shell:NetHood` | Sitios de red |
| `shell:PrintHood` | Impresoras |
| `shell:Quick Launch` | Quick Launch toolbar (vieja) |
