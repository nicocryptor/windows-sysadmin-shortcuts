# Atajos de teclado de Windows

Los atajos del SO. Si trabajas en una laptop con trackpad, esto te salva.

## Bloqueo y sesion

| Atajo | Que hace |
|---|---|
| `Win + L` | Bloquea la estacion. Lo mas importante de toda la lista. |
| `Ctrl + Alt + Del` | Pantalla de seguridad (cambiar contraseña, task manager, cerrar sesion) |
| `Ctrl + Shift + Esc` | Task Manager directo, sin pasar por la pantalla de seguridad |
| `Alt + F4` (sobre el escritorio) | Menu de apagado / cerrar sesion / reiniciar clasico |

## Navegacion entre ventanas

| Atajo | Que hace |
|---|---|
| `Alt + Tab` | Cambiar entre ventanas |
| `Win + Tab` | Task View |
| `Win + D` | Mostrar / ocultar escritorio |
| `Win + ,` (mantener) | Peek al escritorio sin minimizar |
| `Win + Home` | Minimiza todas menos la actual |
| `Win + Shift + M` | Restaura las minimizadas |
| `Win + Flecha` | Snap de ventana (izq/der/arriba/abajo) |
| `Win + Shift + Flecha` | Mueve la ventana al monitor del costado |

## Apps y barra de tareas

| Atajo | Que hace |
|---|---|
| `Win + 1` (2, 3...) | Abre o foco a la app pinneada en esa posicion de la taskbar |
| `Win + Shift + 1` | Abre una nueva instancia de la app pinneada en esa posicion |
| `Win + Ctrl + Shift + 1` | Abre la app pinneada en esa posicion como administrador |
| `Ctrl + Shift + clic` (en taskbar) | Abre la app como administrador |
| `Shift + clic` (en taskbar) | Abre una nueva instancia |

## Accesos directos del SO

| Atajo | Que abre |
|---|---|
| `Win + R` | Cuadro Ejecutar |
| `Win + E` | Explorador de archivos |
| `Win + I` | Configuracion de Windows |
| `Win + X` | Menu de power user (Admin de equipos, Visor de eventos, etc.) |
| `Win + Pause` | Informacion del sistema |
| `Win + .` | Panel de emojis y simbolos |
| `Win + V` | Historial de portapapeles (si esta activado) |
| `Win + Shift + S` | Recorte de pantalla |
| `Win + PrtScn` | Captura completa, se guarda en Capturas |
| `Win + Alt + R` | Grabador de pantalla (Win 11) |

## Explorador de archivos

| Atajo | Que hace |
|---|---|
| `Alt + D` | Foco a la barra de direccion |
| `Ctrl + L` | Igual que Alt+D, mas comodo |
| `F2` | Renombrar |
| `Shift + F10` | Click derecho desde teclado |
| `Alt + Enter` | Propiedades del item seleccionado |
| `Alt + Flecha izq` | Atras |
| `Alt + Flecha arriba` | Subir un nivel |
| `Ctrl + Shift + N` | Nueva carpeta |
